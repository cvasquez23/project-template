## Notes

### Console Commands

- Start watching sass and start dev server.

```console
gulp watch
```

- Purge unused CSS

```console
gulp purgecss
```

- Minify CSS

```console
gulp styles
```

- Minify JS

```console
gulp scripts
```

- Minify HTML

```console
gulp pages
```
